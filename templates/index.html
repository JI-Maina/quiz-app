Welcome! This is a real project with a frontend (HTML + CSS) and a backend (Flask API). The pages work — your job is to finish the API endpoints in app.py.

How the project flows

1

Create

Add World Cup questions through the form. Frontend sends data to POST /api/questions.

Go to Create
2

Fetch & Play

Play page loads questions from GET /api/questions, then you answer and submit.

Go to Play
3

Submit & Results

Answers go to POST /api/play/submit. Scores appear on the Results page.

Go to Results

API endpoints to complete

Method URL What it does
GET /api/questions Fetch all questions (no answers)
POST /api/questions Create a new question
POST /api/play/submit Submit answers and get score
GET /api/results Fetch all past results
GET /api/results/<id> Fetch one result
